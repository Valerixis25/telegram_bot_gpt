# 🤖 Multi-Functional ChatGPT Telegram Bot

Цей Telegram-бот поєднує можливості штучного інтелекту **ChatGPT** та інтерактивні розважальні інструменти. Бот підтримує гнучкі діалоги, ігри, переклади та генерацію фактів у зручному інтерфейсі месенджера.

---

## 🚀 Основні функції

Проєкт підтримує 5 ключових команд:

* **`/random`** — видає випадковий цікавий та науковий факт.
* **`/gpt`** — запускає прямий текстовий діалог із моделлю ChatGPT.
* **`/talk`** — імітує розмову з відомою особистістю. Користувач може обрати персонажа із запропонованого списку, після чого ШІ почне спілкуватися у його фірмовому стилі та манері.
* **`/quiz`** — інтерактивна вікторина. Користувач обирає тематику запитань, а бот веде облік правильних та неправильних відповідей.
* **`/translate`** — швидкий перекладач текстових повідомлень. Користувач вказує текст та обирає цільову мову зі списку доступних варіантів.

---

## 🛠 Технологічний стек

* **Мова програмування**: Python 3.9+
* **Робота з API Telegram**: `aiogram` або `python-telegram-bot`
* **Ядро штучного інтелекту**: [OpenAI API](https://platform.openai.com) (модель ChatGPT)
* **Додаткові інструменти**: `pydantic` (валідація), `python-dotenv` (керування конфігурацією)

## 📦 Встановлення та запуск

### 1. Клонування репозиторію
```bash
git clone https://github.com
cd your-repo-name
```

### 2. Налаштування віртуального оточення
```bash
python -m venv venv
# Для Windows:
venv\Scripts\activate
# Для macOS/Linux:
source venv/bin/activate
```

### 3. Встановлення залежностей
```bash
pip install -r requirements.txt
```

### 4. Налаштування змінних оточення
Створіть файл `.env` у кореневій директорії проєкту за шаблоном:
```env
TELEGRAM_BOT_TOKEN=your_telegram_bot_token_here
OPENAI_API_KEY=your_openai_api_key_here
```
> *Токен для бота можна отримати у офіційного батька ботів [@BotFather](https://t.me), а ключ ШІ — на платформі [OpenAI Developers](https://platform.openai.com).*

### 5. Запуск бота
```bash
python main.py
```

## 📝 Приклади використання команд у чаті

1. `/random` ➔ *Бот виводить рандомний цікавий факт
2. `/gpt` ➔ *Бот переходить у режим асистента
3. `/talk` ➔ *Бот показує інлайн-кнопки: `[Курт Кобейн]` `[Фрідріх Ніцше]` `[Стівен Гокінг]`.* Після вибору героя ШІ відповідатиме так, нибі ви спілкуєтеся з самим героєм.
4. `/quiz` ➔ *Користувач обирає тему `[Математика]`, бот надсилає запитання.*
5. `/translate` ➔ *Оберіть мову `[Англійська]` ➔ Введіть текст ➔  Отримайте миттєвий результат.*
